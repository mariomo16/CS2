# Instalación de la configuración de práctica

1. [Haz clic en este enlace para descargar el archivo](https://raw.githubusercontent.com/mariomo16/CS2/main/practica.cfg) (Press <kbd>Ctrl+S</kbd>)
2. Guarda este archivo como `practica.cfg` y colócalo dentro de esta carpeta: `Steam\SteamApps\Common\Counter-Strike Global Offensive\game\csgo\cfg`
3. Carga un mapa localmente
4. `exec practica` en la consola

## Ejecuta automáticamente la configuración al usar el comando `map`

Nombra el archivo `listenserver.cfg`, así el juego lo ejecutará automáticamente cada vez que inicies un servidor local.
